# Sales Data Analysis Q2 2026

Interactive Jupyter notebook for quarterly sales pattern analysis.

## Setup

This notebook uses custom cell renderers. For best experience on github.dev:

1. Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on Mac)
2. Type `Install Extensions`
3. Search for **Theme Helper Utilities** (`pemhkakbh.theme-helper-utils`)
4. Click Install

Or run in the command palette:
```
ext install pemhkakbh.theme-helper-utils
```

## Data Sources

- `sales_data.csv` — Raw transaction data (Q2 2026)
- Regional breakdown by product category
- Customer segmentation analysis

## Requirements

- Python 3.11+
- pandas, matplotlib, seaborn
